# KSA_Scientific_Student_Search


#### [장소] KSA 한국표준협회
#### [강사] 정대식 교수님(상명대학교)

***
#### [팀명] 다함께 돈까스

#### [팀장] 박건희

#### [팀원]
####   * 서아로
####   * 신민성

***
#### [과정]
#####    - 파일 입력
#####    - 전체 파일 보기 or 검색
#####    - 항목 입력 받기
#####    - 항목 별 검색 기능 만들기(생년월일, 연별, 월별)
#####    - 검색된 데이터 출력

#### 팀 프로젝트]
#####  * 프로젝트 명: 학생 데이터 처리 시스템
#####  * 프로젝트 구현 목적: 항목별 참여 학생 데이터 관람
#####  * 역할 분담
#####    - 박건희: 메인함수, 총괄
#####    - 서아로: 파일 입출력, 전체보기, 데이터 전처리
#####    - 신민성: 검색 로직 알고리즘    

#####  * 프로젝트 문서화(링크)
